## Sobre

A ideia é fazer toda a modelagem de um sistema de biblioteca escolar antes de partir pra parte de programação.

Sistema pra o controle de uma biblioteca escolar: cadastro de livros, empréstimos, devoluções, reservas e multas por atraso. Atende quatro tipos de usuário: **aluno**, **professor**, **bibliotecário** e **administrador**.

## O que temos aqui

- **MER Conceitual e Lógico** — banco de dados
- **Caso de Uso**
- **Diagrama de Classes**
- **Atividades** — empréstimo, devolução e reserva
- **Sequência** — empréstimo
